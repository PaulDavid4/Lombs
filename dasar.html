Apa itu HTML ?

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan
untuk membuat halaman web dan aplikasi web.
HTML terdiri dari serangkaian elemen dan tag yang menentukan struktur dan konten halaman web
